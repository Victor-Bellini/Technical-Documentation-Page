HTML

HTML (HyperText Markup Language) allows us to insert elements and objects in our workspace, such as lists, tables, images, etc.

The example below shows us the code necessary to insert a paragraph with the text "Hello World".

< p > Hello World < / p >
CSS

Aside from the HTML, the CSS allows us to style our elements, working in a different structure and file.

This is how we say we want our "Hello World" text in black.

p {

color: black;

}
